 Функции и параметры
при разработке большества приложений их разделяют на функциональные модули.
В c# приложение состоит из классов, которые содержат иминованые блоки, кода называемые методы.(функции это часть класса, который может осуществелять действие или вычетать действие).
использование методов
Метод - набор операторов которые выполняет месте для создание метода надо создать ему имя и определить список метода и тип 
Для вызова метода другого класса  необходимо чтобы она был обявлин как публик вызов осуществляется по имени и класса если методов не void, то не обходимо вернуть значение соотвественого типа о
Оператор retern останавливает метод и передает направление вызовшиму метод 
Каждый метод имеет каждый метод имеет локальных переменых они видны только в нем и при заверщение метода они уничтожаются для того что переменная была видна в несколько метода класса надо объявить в поле не метода но внутри класса 
Для не void методов необходима возрощять методы каждый "" путь выполнения метода должен возратся оператором retern
Для void метода оператор не обязательно.

Пользование параметров
Параметры позволяют передавать информацию с одного метода в другой При объявление метода можно задать ему параметров, если список пустой, то означает то метод не имеет параметров.
C# существует 3 варианта передачи параметров ;
По значению ( изменение значение параметра методе не влияет на влияет на значение в вызвашем методе ) 
по ссылки ( необходима указывать клювле слово ref)
До вызова метода необходима инцелизировать переменную в 
В с# позволяет мехонизм передачи списка параметром длины для этого используется паразм.
Модицикаторы доступа 
Ключевые слова которые задают определеный объявленый уровень доступности члена или типа 
7 уровней спец возможности модификатора доступа 
public: доступ не ограничен
protected: доступ ограничен содержащим классо
интернал: доступ ограничен 
прайвад проктетор: доступ ограничен содержащим типом  
файл: объявлен тип отображается только в текущем файле
модификатор простраства указывает что отсуствищую элемент 
модификатор оси позволяет указать что метод указывает сихроными 
модификатор ст можно добавить локальную функцию, используется для объявление статического члена, пренадлежащего собственого а не другому.
